The figure number is reflected in the name of the zip file containing this data.

The data file Port4MeanUnc_ErrorInSingleResult_MC_dB.txt has the following format.
The data file has seven columns:
Column 1: Frequency in GHz
Column 2: Nominal solution (no bias) in db
Column 3: Monte-Carlo estimate (includes statistical bias) in dB
Column 4: Monte-Carlo estimate + standard uncertainty in dB
Column 5: Monte-Carlo estimate - standard uncertainty in dB
Column 6: Monte-Carlo upper 95% confidence interval in dB
Column 7: Monte-Carlo lower 95% confidence interval in dB

The data files S11_dB-n.txt have the following format.
Each data file has two columns:
Column 1: Frequency in GHz
Column 2: Magnitude of S11 in dB



Files with the extension .plt are Easy Plot menus used to generate the figure indicated in the name of the zip file.
