The figure number is reflected in the name of the zip file containing this data.

Each data file has five columns:
Column 1: Frequency in GHz
Column 2: Monte-Carlo estimated std unc in dB
Column 3: Sensitivity-analysis estimated std unc in dB
Column 4: 95% upper confidence interval for convergence of Monte-Carlo std unc in dB
Column 5: 95% lower confidence interval for convergence of Monte-Carlo std unc in dB

Files with the extension .plt are Easy Plot menus used to generate the figure indicated in the name of the zip file.
